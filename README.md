# MiniGPT

A Mini GPT-like Language Model built from scratch using PyTorch.

## Project Overview

This project demonstrates the core concepts behind Large Language Models (LLMs) and GPT architectures.

The model was developed step-by-step to understand:

- Tokenization
- Vocabulary Creation
- Encoding & Decoding
- Embeddings
- Self-Attention
- Transformer Layers
- Training Process
- Text Generation

## Technologies Used

- Python
- PyTorch
- NLP
- Transformers

## Features

- Character-level Tokenization
- Vocabulary Mapping
- Embedding Layer
- Multi-Head Self Attention
- Transformer Encoder
- Language Model Head
- Training Loop
- Next Character Prediction
- Text Generation

## Workflow

1. Load Dataset
2. Build Vocabulary
3. Encode Text into Tokens
4. Create Embeddings
5. Apply Self-Attention
6. Pass Through Transformer Layer
7. Generate Logits
8. Calculate Loss
9. Train Model
10. Generate Text

## Sample Dataset

```text
hello world
hello ai
ai is awesome
machine learning is fun
artificial intelligence is powerful
```

## Example Output

Input:

```text
hell
```

Output:

```text
hello
```
